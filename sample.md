# 示例文件

本文档为 MD → DOCX 转换器的示例文件，用于展示各类 Markdown 元素的转换效果。

## 项目简介

本项目旨在提供一个便捷的 Markdown 到 Word 文档转换工具，支持自定义样式配置，满足学术论文、技术报告等场景的排版需求。

工具的核心功能包括：

- 标题自动编号（支持中文编号格式）
- 公式编号与章节关联
- 表格样式自定义（网格线/三线表）
- 页眉页脚与页码设置

## 排版规范说明

### 标题层级

文档支持六级标题，其中一级标题采用中文大写编号（一、二、三），二级标题采用阿拉伯数字编号（1.1, 1.2），三级标题采用三位编号（1.1.1, 1.1.2）。

### 正文格式

正文默认采用仿宋字体，字号为小四（12pt），行距为单倍行距。首行缩进两字符，段前段后间距可通过配置文件调节。

$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$ (3)

## 列表示例

### 无序列表

项目采用的技术栈如下：

- 前端框架
  - CustomTkinter（桌面端）
  - 响应式布局设计
- 后端处理
  - python-docx 文档生成
  - PyYAML 配置管理
- 文档转换
  - Markdown 解析引擎
  - Word XML 操作

### 有序列表

安装步骤：

1. 安装 Python 3.8 或更高版本
2. 克隆项目代码仓库
3. 安装依赖包：pip install -r requirements.txt
4. 运行程序：python main.py
5. 选择 Markdown 文件并配置参数
6. 点击"开始转换"生成 Word 文档
